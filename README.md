# Тестовая форма отзыва «Ваш Контроль»

Статический прототип формы отзыва для `vashkontrol.ru`. Проект остаётся без сборщика, зависимостей и backend.

## Как открыть

Запускайте проект через локальный сервер, иначе браузер не загрузит JSON-справочники через `fetch()`:

```bash
uv run python -m http.server 8000
```

После запуска откройте:

```text
http://localhost:8000
```

## Справочники

Данные вынесены из `script.js` в отдельные файлы:

- `data/regions.json` — регионы и города для поля региона;
- `data/services.json` — услуги, категории, коды и синонимы;
- `data/organizations.json` — МФЦ, ведомства, адреса, региональная принадлежность и связи с услугами.

Федеральные организации помечаются через `regionScope: "federal"`.

## Поиск и фильтрация

- Поиск разделён на вкладки `Организации` и `Услуги`.
- Во вкладке `Организации` поиск показывает только организации.
- Во вкладке `Услуги` поиск показывает только услуги.
- Регион теперь жёстко фильтрует выдачу организаций.
- Если выбран город, сначала идут организации этого города, затем остальные организации региона.
- Организации других регионов скрываются.
- Федеральные организации показываются только при типе получения `Онлайн`.

## Что работает

- Кастомный поиск региона с клавиатурой: `ArrowDown`, `ArrowUp`, `Enter`, `Escape`.
- Выбор точной пары `услуга + организация`.
- Кастомный светлый календарь с ограничением: не будущее и не старше 3 лет.
- Пять обязательных оценок по шкале 5 звёзд.
- Необязательные комментарий, фото и ссылка на видео.
- Видимый юридический блок `Хочу получить официальный ответ`.
- Сохранение черновика в `localStorage`.
- Локальное подтверждение отправки и запись данных отзыва в `console.log`.

Отзыв нельзя отправить, пока не выбрана конкретная услуга и конкретная организация.
