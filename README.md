# Тестовая форма отзыва «Ваш Контроль»

Статическая форма отправки отзывов для демонстрации сценария. Дизайн сделан ближе к текущему стилю: простая шапка, белые карточки, зелёные акценты, минимум декоративных блоков.

## Как открыть

Откройте `index.html` в браузере. Сборка и сервер не нужны.

## Что работает

- локальный поиск услуги и организации по обычным словам;
- отдельные режимы поиска: «Не знаю», «Получал в МФЦ», «Получал в ведомстве»;
- учёт региона, этапа оценки и типа получения услуги;
- выбор связки «услуга + организация»;
- обязательные оценки по 5 параметрам;
- комментарий остаётся необязательным;
- фото-превью и проверка ограничения 5 файлов по 5 МБ;
- ссылка на видео с проверкой URL;
- юридический блок «Хочу получить официальный ответ» всегда виден;
- сохранение черновика в `localStorage`;
- отправка с локальным подтверждением и сохранением данных в `console.log`.

## Логика поиска

Прототип не подключается к внешним сервисам. В `script.js` есть моковые справочники услуг и организаций. Поиск работает локально:

1. нормализует запрос: нижний регистр, замена `ё` на `е`, удаление лишней пунктуации;
2. ищет совпадения по названию услуги, простым синонимам, ведомству, городу, адресу и региону;
3. строит пары только там, где организация действительно оказывает выбранную услугу;
4. повышает релевантность МФЦ или ведомств в зависимости от выбранного режима;
5. для онлайн-сценария немного выше поднимает ведомства;
6. если пользователь ничего не нашёл, открывает отдельное сообщение об ошибке в справочнике.

Главное правило: отзыв нельзя отправить, пока не выбрана конкретная услуга и конкретная организация.
